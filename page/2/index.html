Linux安装tomcat、mysql

Linux安装tomcat、mysql linux上安装mysql5.7 1、下载tar包，这里使用wget从官网下载 wget https://dev.mysql.com/get/Downloads/MySQL-5.7/mysql-5.7.22-linux-glibc2.12-x86_64.tar.gz 2、将mysql安装到/usr/local/mysql下 解压 tar -xvf mysql-5.7.22-linux-glibc2.12-x86_64.tar.gz 移动 mv mysql-5.7.22-linux-glibc2.12-x86_64 /usr/local/ 重命名 mv /usr/local/mysql-5.7.22-linux-glibc2.12-x86_64 /usr/local/mysql 3、新建data目录 mkdir /usr/local/mysql/data 4、新建mysql用户、mysql用户组 # mysql用户组 groupadd mysql # mysql用户 useradd mysql -g mysql 5、将/usr/local/mysql的所有者及所属组改为mysql chown -R mysql.mysql /usr/local/mysql 6、配置 /usr/local/mysql/bin/mysql_install_db –user=mysql –basedir=/usr/local/mysql/ –datadir=/usr/local/mysql/data 如果出现以下错误： 2018-07-14 06:40:32 [WARNING] mysql_install_db is deprecated.
